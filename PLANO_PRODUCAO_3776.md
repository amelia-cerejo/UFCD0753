# Plano de producao da UFCD 3776

## Objetivo deste plano

Organizar a producao dos materiais da UFCD 3776 sem sair do referencial.

## Principio de trabalho

Cada material deve responder a uma pergunta simples:

> Este conteudo ajuda o formando a utilizar aplicacoes de folha de calculo ou de base de dados, tal como previsto no referencial?

Se a resposta for nao, o conteudo deve ficar de fora.

## Sequencia recomendada

### Fase 1 - Base de controlo

- Confirmar referencial.
- Confirmar identidade visual.
- Organizar imagens e recursos.
- Criar uma pagina-piloto.

### Fase 2 - Conteudos de folha de calculo

Materiais a produzir:

- caracteristicas da folha de calculo;
- vantagens e desvantagens;
- criacao, gravacao e edicao;
- formatacao;
- graficos com modelos pre-definidos;
- preparacao de conteudos para a Internet.

### Fase 3 - Conteudos de base de dados

Materiais a produzir:

- caracteristicas da base de dados;
- vantagens e desvantagens;
- modo de funcionamento;
- componentes de uma base de dados;
- manutencao de bases de dados;
- publicacao de conteudos para a Internet.

### Fase 4 - Atividades

As atividades devem ser praticas e simples:

- criar e editar uma folha de calculo;
- formatar uma tabela;
- criar um grafico a partir de dados fornecidos;
- reconhecer componentes de uma base de dados;
- interpretar relacoes simples entre dados;
- preparar ou publicar conteudos simples para a Internet.

### Fase 5 - Revisao

Antes de fechar qualquer material:

- confirmar se o conteudo esta no referencial;
- verificar se a linguagem e clara;
- verificar se o visual respeita a identidade;
- confirmar que nao foram acrescentados temas avancados;
- rever se existe equilibrio entre explicacao, exemplo e pratica.

## Estado final de producao

1. Site principal com conteudos, atividades, avaliacao e recursos.
2. Conteudos Gamma incorporados nas paginas internas do site.
3. Recursos de folha de calculo disponibilizados em ficheiros e videos.
4. Avaliacoes incorporadas a partir do projeto `avaliacoes-formacao`.
5. Publicacao final no Netlify para utilizacao atraves do Moodle.

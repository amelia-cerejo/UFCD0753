# Identidade visual da UFCD 3776

## Identificacao

**Codigo da UFCD:** 3776  
**Nome da UFCD:** Informatica - folha de calculo e base de dados  
**Area tematica:** Competencias digitais, organizacao de informacao e produtividade

## Perfil dos formandos

Formandos em contexto de aprendizagem inicial ou intermédia, que precisam de ganhar autonomia no uso de ferramentas digitais para organizar, tratar e consultar informacao.

## Personalidade visual

- Moderna
- Tecnologica
- Profissional
- Clara
- Orientada a dados

## Paleta

### Folha de calculo

- Verde principal: `#107C41`
- Verde secundario: `#22C55E`
- Verde claro: `#DCFCE7`

### Base de dados

- Laranja principal: `#F97316`
- Laranja secundario: `#FDBA74`
- Laranja claro: `#FFF7ED`

### Neutros

- Texto principal: `#111827`
- Texto secundario: `#374151`
- Cinzento: `#6B7280`
- Fundo suave: `#F8FAFC`
- Branco: `#FFFFFF`

## Metaforas visuais

- folha de calculo;
- tabela;
- grafico;
- dashboard;
- base de dados;
- relacoes entre tabelas;
- dados organizados que apoiam decisoes.

## Elementos graficos recomendados

- tabelas limpas;
- graficos simples;
- cartoes de indicadores;
- icones lineares;
- ligacoes entre blocos de dados;
- divisao visual clara entre verde e laranja.

## Elementos a evitar

- excesso de efeitos decorativos;
- fundos demasiado escuros;
- paletas monocromaticas;
- imagens sem relacao direta com folha de calculo ou base de dados;
- linguagem visual demasiado infantil.

## Componentes pedagogicos prioritarios

- Conceito
- Exemplo
- Passo a passo
- Experimenta
- Aplica
- Revê

## Tom de comunicacao

Claro, proximo, profissional e orientado a acao.

As frases devem ser curtas e explicar o objetivo de cada passo. A linguagem deve ajudar o formando a perceber o que esta a fazer e porque o esta a fazer.

## Prompt curto da identidade

```text
Identidade visual da UFCD:

UFCD: 3776 - Informatica: folha de calculo e base de dados
Estilo: moderno, tecnologico, profissional, claro e orientado a dados
Paleta: verde para folha de calculo, laranja para base de dados, neutros claros
Metaforas visuais: tabelas, graficos, dashboards, bases de dados e relacoes entre informacao
Elementos graficos: tabelas limpas, graficos simples, icones lineares, blocos relacionados
Tom: claro, didatico, pratico e orientado passo a passo
Evitar: conteudos fora do referencial, excesso decorativo, temas avancados nao previstos

Cria materiais formativos digitais modernos, claros e profissionais, mantendo a separacao visual entre folha de calculo e base de dados.
```

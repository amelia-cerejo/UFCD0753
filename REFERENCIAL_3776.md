**Designacao:** Informatica - folha de calculo e base de dados

## Objetivo

Utilizar aplicacoes informaticas de folha de calculo e de base de dados.

- Formatacao
- Graficos com recurso a modelos pre-definidos
- Preparacao de conteudos para a Internet

### 2. Base de dados

- Caracteristicas
- Vantagens e desvantagens
- Modo de funcionamento
- Componentes de uma base de dados
- Manutencao de bases de dados

## Limites editoriais

Para manter os materiais dentro do referencial, devem ser evitados temas avancados que nao estejam previstos, como:

- programacao VBA;
- Power Query ou automacoes avancadas;
- SQL avancado;
- administracao de servidores de bases de dados;
- modelacao complexa fora do nivel introdutorio;
- analise estatistica avancada.

Estes temas so devem ser usados se forem estritamente necessarios como referencia breve e sempre sem transformar o conteudo central da UFCD.
